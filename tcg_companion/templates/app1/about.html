About the Website

A buddy and I decided to make this website for a class project. One of the use cases of this web application is to store cards into the website to be able to keep track of all of them instead of going through a binder of cards. In addition, this can help give competitive players an advantage on how to improve decks by receiving strategies from Google Gemini. Thanks for checking out our website and good luck catching them all!
